Здесь представлена работа с платой NUCLEO-STM32F411RE.  
Проекты написаны с использованием различных библиотек или без оных.  
Соответственно, в названиях проектов это отображено в виде суффика с названием библиотеки:  
HAL - на библиотеке HAL;  
CMSIS - на библиотеке CMSIS;  
empty - без библиотек, руками, напрямую программируя регистры.  

В каждом проекте имеется свой README файл, с более подробным описанием проекта.
